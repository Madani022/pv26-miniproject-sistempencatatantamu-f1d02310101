# Sistem Pencatatan Tamu - Mini Project PV26

## Deskripsi Singkat
Sistem Pencatatan Tamu adalah aplikasi GUI berbasis *desktop* yang dirancang untuk mempermudah pencatatan dan pengelolaan data tamu atau pemohon di instansi layanan publik. Aplikasi ini dibangun dengan mengimplementasikan arsitektur *Separation of Concerns* (SoC) untuk memisahkan logika UI, Controller, dan Database.

Aplikasi ini dibuat sebagai pemenuhan Tugas Mini Project mata kuliah Pemrograman Visual.
